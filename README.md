# Mapsgram Legal

Mapsgram uygulamasının resmi yasal dokümanları. GitHub Pages ile yayında.

## Sayfalar

- **Ana sayfa:** https://githubbkadir.github.io/mapsgram-legal/
- **Gizlilik Politikası:** https://githubbkadir.github.io/mapsgram-legal/privacy.html
- **Kullanım Koşulları:** https://githubbkadir.github.io/mapsgram-legal/terms.html

## GitHub Pages'i aktif etme

1. Bu repo'nun **Settings** sekmesine git
2. Sol menüden **Pages**
3. **Source:** Deploy from a branch → **Branch: main, folder: / (root)** → Save
4. Birkaç dakika sonra URL aktif olur (yukarıdaki adresler çalışır)

## Placeholder'ları doldur

`privacy.html` ve `terms.html` içinde sarı renkli `[KÖŞELİ PARANTEZ]` yerleri var. Bunları kendi bilgilerinle değiştir:

- `[TAM ADIN/ŞİRKET ADI]`
- `[POSTA ADRESİN]`
- `[DESTEK EMAIL]` (örn. `destek@mapsgram.com`)
- `[ŞEHİR]` (yetkili mahkeme — örn. İstanbul)

## Play Store'a verme

Google Play Console → Mapsgram → **App content** → **Privacy policy** → URL gir:
```
https://githubbkadir.github.io/mapsgram-legal/privacy.html
```
